# WID-005 — 店舗別時間帯比較

## 1. 概要

`UnifiedStoreHourlyWidget` に full ctx を passthrough し、店舗別の時間帯比較を描画する registry entry（registryChartWidgets.tsx:101-107）。

## 2. Data Acquisition Path

- registry 行では `ctx` を full passthrough
- 実際のデータ取得・描画は全て `UnifiedStoreHourlyWidget` 内部に委譲
- `isVisible` に **関数参照** `isStoreComparisonVisible` を指定

## 3. Context Fields Consumed — 解説

- registry 行からの直接参照 0（full ctx passthrough）
- 実 ctx 依存は子 `UnifiedStoreHourlyWidget` 側に散在

## 4. Composition

- 直接の子: `UnifiedStoreHourlyWidget`（`./UnifiedAnalyticsWidgets` から import、:8）
- WID-004 と同一 source file の隣接 widget

## 5. Visibility

- 条件: `isVisible: isStoreComparisonVisible`（`widgetVisibility.ts` の関数参照）
- predicate 本体は本 spec では未追跡

## 6. Invariants

- 本 widget 自身は数学的不変条件を持たない
- 店舗間比較の集計不変は子経由で timeSlot / storeDaily lane に従属
