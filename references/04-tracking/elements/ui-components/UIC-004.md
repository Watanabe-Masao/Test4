# UIC-004 — Chart 共通 Wrapper

## 1. 概要

`ChartCard` は **chart 全種類で共有される容器 component** で、header（タイトル / 説明 / Explain ボタン）+ toolbar（凡例 / モード切替）+ body（chart 本体）+ state overlay（loading / empty / error）を統一した shell を提供する。EChart 等の chart 本体を `children` で受け取る composition pattern。

## 2. Props Contract

主要 props（line 36〜）:

- `title`: chart タイトル
- `description`（option）: 補足説明
- `onExplain`（option）: 説明 modal 起動 callback
- `toolbar`（option）: ReactNode（凡例 / モード切替等）
- `state`（option）: `'loading' | 'empty' | 'ready' | 'error'`（state overlay 表示）
- `errorMessage`（option）: error state 時の message
- `children`: chart 本体（通常は `EChart` / `Recharts` 等）
- `height`（option）: 固定高さ

## 3. State Ownership

- 内部 state なし（state は props 駆動）
- 実際の chart 本体（EChart 等）は children で受け取り、外部 state 管理

## 4. Hook Dependencies

- なし（pure shell）

## 5. Side Effects

- `onExplain` callback 経由のみ
- chart 本体の DOM access / canvas / portal は children 側に閉じる

## 6. State Coverage

| state | 振る舞い |
|---|---|
| empty | `<ChartEmpty>` overlay 表示（children skip）|
| loading | `<ChartLoading>` overlay（skeleton）|
| ready | children 描画 + header / toolbar |
| error | `<ChartError message={errorMessage}>` overlay |

UIC-005（ChartLoading / ChartError / ChartEmpty）と pair で運用。
