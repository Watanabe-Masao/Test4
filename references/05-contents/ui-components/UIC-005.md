# UIC-005 — Chart 状態 Overlay (Loading / Error / Empty)

## 1. 概要

`ChartLoading` は **chart の loading 状態を表示する skeleton/spinner component** で、`ChartError` (line 34) と `ChartEmpty` (line 42) と同 module で pair 運用される。`ChartCard` (UIC-004) の `state` props と連動して overlay 表示される。

## 2. Props Contract

3 component の合計 props:

- `ChartLoading({ height? })` — height (optional) で chart 領域に合わせた skeleton
- `ChartError({ message: string, height? })` — error message + 再試行 hint
- `ChartEmpty({ message?: string, height? })` — empty state placeholder

すべて `height` で chart 本体と同じ垂直スペースを占有し、layout shift を防ぐ。

## 3. State Ownership

- なし（pure presentation）

## 4. Hook Dependencies

- なし

## 5. Side Effects

- なし
- DOM 直接 access / animation 起動なし（CSS animation は styled-components）

## 6. State Coverage

本 component 自身が state overlay の正本:

| 表示する状態 | 役割 |
|---|---|
| Loading | データ取得中の skeleton 表示 |
| Error | データ取得失敗 + 再試行 hint |
| Empty | データ 0 件 / フィルタ結果なし |

各 chart の `state` props と 1:1 対応。
