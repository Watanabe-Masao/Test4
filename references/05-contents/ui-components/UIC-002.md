# UIC-002 — KPI カード

## 1. 概要

`KpiCard` は **pure 表示** React component で、単一 KPI（指標 + 値 + 単位 + 変化率 + warning + display mode）を統一フォーマットで表示する公開契約 component。`KpiDisplayMode` (`authoritative` / `reference` / `hidden`) と `KpiWarningInfo` で表示ニュアンスを切替可能。

## 2. Props Contract

主要 props（line 33〜）:

- `label`: KPI 名
- `value`: 数値
- `unit`（option）: 単位記号
- `delta`（option）: 変化率
- `warning?: KpiWarningInfo` — `{ severity, message }` で警告表示
- `displayMode?: KpiDisplayMode` — `'authoritative'` / `'reference'` / `'hidden'`（default `'authoritative'`）
- `onClick`（option）: クリック handler（callback）
- `onExplain`（option）: 説明 modal 起動 callback (`metricId` 等)

## 3. State Ownership

- 内部 state なし（pure presentation）

## 4. Hook Dependencies

- なし（直接 hook 使用なし）
- `displayMode === 'hidden'` で render skip

## 5. Side Effects

- `onClick` / `onExplain` 経由で外部 callback 起動のみ
- DOM 直接 access なし

## 6. State Coverage

| state | 振る舞い |
|---|---|
| empty | `value` が `null` / `undefined` で `-` 等の placeholder 表示 |
| ready | 数値表示 + delta + warning |
| loading | `value === null` を上位 hook で skeleton 化（KpiCard 自体は同期）|
| error | `warning` field で error 表示 |
