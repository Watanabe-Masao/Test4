# WID-032 — 予算と実績（Insight）

## 1. 概要

`BudgetTabContent` に `insightData` / `result` / `onExplain` を forward する Insight page widget（Insight/widgets.tsx:14-24）。`isVisible` + render 内で **二重 null check**（`insightData != null`）。

## 2. Data Acquisition Path

- `isVisible: (ctx) => ctx.insightData != null` で gate
- render 内で `if (!ctx.insightData) return null`（isVisible と重複）
- 通過時に `<BudgetTabContent d={ctx.insightData} r={ctx.result} onExplain={ctx.onExplain} />`

## 3. Context Fields Consumed — 解説

- 3 field touch
- `insightData` は UnifiedWidgetContext の page-local optional（inquiry/04 §B-2）

## 4. Composition

- 直接の子: `BudgetTabContent`（`./InsightTabBudget` から import、:9）

## 5. Visibility

- 条件: `(ctx) => ctx.insightData != null`
- render 内でも再 check（二重）

## 6. Invariants

- 本 widget 自身は数学的不変条件を持たない
- 予算・実績の計算不変は子 `BudgetTabContent` + `useInsightData`（inquiry/06 §A-1）に従属

## 7. Co-Change Impact

- `InsightData` 型の変更
