## 命名規約

```
NN-<slug>.md
```

- `NN`: 2 桁の連番（`01` / `02` / ...）
- `<slug>`: kebab-case の短い説明
- 例: `01-comparison-math-in-presentation.md`

## 章テンプレート

各棚卸しファイルは以下の構造を持つ:

```markdown
# NN-<slug> — <タイトル>

## 事実

<観測された事実。ファイル・行・コマンド・エラーで記録>

## 影響

<この事実がなぜ問題か / 何を壊すか>

## 設計案

<対処の方向性 / 残すべき論点>
```
