## 本 project の目的

整合性 framework に Phase R (Framework Reset) を施し、その上で Phase H (Horizontal Expansion) を実施する。前駆 project `canonicalization-domain-consolidation` の Phase A〜I 完遂で発見された **13 dimension の構造的 / 制度的 gap** を institutional answer に転換する。

## scope

### 含む

- **Phase R**: 構造的問題 A/B/C (双方向契約 / 時間軸 / 3-zone) + 制度的問題 D/E/F (cross-domain framework / decision artifact standard / dogfooding) の institutional answer
- **Phase H**: wasm + charts + hooks の horizontal expansion (Phase R framework 上で)
- **Phase I 拡張**: §P8/§P9 / canonicalization-checklist.md の 3-zone 化 + 前駆 project archive

### 含まない (nonGoals)

- §P8/§P9 の **修正** (拡張は可、変更不可)
- 業務 logic の変更 (registry+guard 整備のみ)
- active overlay 自動切替 (人間判断)
- 第 5 の柱 (Project Lifecycle Governance) のすべて (本 project scope 外、後続 project)
